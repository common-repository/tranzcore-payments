In a few simple steps you can start accepting mobile money payments with TranzCore Payments on your WordPress site.

== Description ==
Shopping cart software modules for online stores offer a quick and easy integration process as well as smooth shopping experience for your customers.
This payment module extends WooCommerce and allows you to accept payments Visa/MasterCard, MTN Mobile Money, Orange Money, Nextel Possa etc.

Your customers will be redirected to the \"Checkout Result\" page after payment. This page shows them the details of the transaction (the item that they just paid for).

The transaction info is also captured in the orders menu of the plugin. You can view all the payments you received from your WordPress admin dashboard.

TranzCore Payments is available in:

* Cameroon

= Features =

* Quick installation and setup.
* Easily take payment for a service from your site via TranzCore Payments.
* Sell files, digital goods or downloads using your TranzCore Payments merchant account.
* Sell music, video, ebook, PDF or any other digital media files.
* Allow users to automatically download the digital file after the purchase is complete.
* View purchase orders from your WordPress admin dashboard.
* Accept donation on your WordPress site for a cause.
* Ability to collect billing and shipping address of the customer.
* Option to accept custom donation amount via TranzCore Payments payment gateway.

The setup is very easy. Once you have installed the plugin, all you need to do is enter your TranzCore Payments API credentials in the plugin settings and your website will be ready to accept mobile money payments.

You can run it in test mode by specifying test API keys in the plugin settings.

== Installation ==
The easiest way to install TranzCore Payments is to search for it via your site’s Dashboard.

= Step-by-step instructions =

1. Log in to your site’s dashboard (e.g. www.yourdomain.com/wp-admin).
2. Click on the “Plugins” tab in the left panel, then click “Add New”.
3. Search for “TranzCore Payments” and find our plugin near the top. Look for the author “TranzCore Co Ltd”.
4. Install it by clicking the “Install Now” link.
5. When installation finishes, click “Activate Plugin”.
6. Open the settings page for WooCommerce and click the “Payment Gateways” tab
7. Click on the sub tab for \'TranzCore Payments\'
8. Configure your TranzCore Commerce Gateway settings

== Frequently Asked Questions ==

= Can I have multiple payment buttons on a single page? =

Yes, you can have any number of buttons on a single page.

= Can I use it in a WordPress Widgets? =

Yes, you can.

= Can I change the button label? =

Yes, you can specify the title nn your TranzCore configuration interface.

= Will the customers receive an email after purchase? =

Yes, Wordpress has built in notifitaction on transaction status. Clients will get an email update on the transaction.

= Can i use my account on multiple websites?

Yes. Go to you TranzCore merchant account and select the business tap. there you can add multiple websites.

= Can It be tested before going live? =

Yes, please visit TranzCore Payments Payments > Settings screen for options.
